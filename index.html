Native macOS clipboard tray

Insert

A fast clipboard history that lifts from the bottom of your screen, keeps the latest item selected, and gets out of your way when you copy.

Download GitHub
Search Find text, links, files, images, PDFs, HTML, and rich text from your clipboard history.
Keyboard first Arrow through cards, press Enter or Command+C to copy, and Backspace to delete.
Quiet settings Customize the global shortcut, hide the Dock icon, and launch at login.

Bottom tray

Your clipboard, one shortcut away.

Insert keeps a compact card history below your current work. The newest item is selected automatically, so most copies are a single keystroke.

Build or install

Download the latest DMG from GitHub Releases, or build locally from source with the included Makefile.

make run
make dmg
